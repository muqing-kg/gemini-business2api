# 节点管理功能更新日志

## 新增功能

### 🎯 代理节点管理
- **节点 CRUD**：完整的增删改查功能
- **订阅导入**：支持 Clash 订阅链接导入
- **YAML 导入**：支持粘贴 YAML 配置导入节点
- **节点统计**：自动记录成功/失败次数，智能选择最优节点
- **批量操作**：支持全选、批量删除
- **搜索排序**：按名称、成功率、次数排序

### 🔧 Clash 集成
- **自动管理**：集成 mihomo (Clash Meta) 核心
- **配置预览**：实时预览运行时 YAML 配置
- **语法高亮**：使用 highlight.js 提供专业的 YAML 高亮
- **健康检测**：自动测试节点延迟和可用性
- **智能切换**：根据成功率自动选择最优节点

### 🎨 UI 优化
- **现代化界面**：全新的节点管理页面
- **代理控制面板**：可视化配置代理端口和开关
- **实时状态**：节点启用状态、成功率可视化
- **响应式设计**：适配各种屏幕尺寸

## 技术实现

### 后端
- `core/node_manager.py` - 节点管理核心逻辑
- `core/clash_manager.py` - Clash 代理管理
- `core/node_stats.py` - 节点统计功能
- 新增 API 端点：`/api/admin/nodes/*`

### 前端
- `frontend/src/views/Nodes.vue` - 节点管理页面
- `frontend/src/api/nodes.ts` - 节点 API 封装
- 集成 `highlight.js` 用于 YAML 语法高亮

## 使用说明

1. 访问管理面板 → 节点管理
2. 点击"导入节点"导入订阅或 YAML 配置
3. 在"代理控制"面板启用代理总开关
4. 系统自动选择最优节点用于账号注册和刷新

## 依赖更新

- 新增：`highlight.js` - YAML 语法高亮
- 新增：`mihomo.exe` - Clash Meta 核心（需单独下载）
